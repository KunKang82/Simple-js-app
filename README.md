_This is a small pokemon web application that loads data from an external public API and enables the viewing of data points in detail built with:_

* HTML
* CSS
* Javascript
* JQuery
* Bootstrap
* [Pokemon API](https://pokeapi.co/api/v2/pokemon/?limit=150)
* Ajax
* Fetch polyfill
* Promise polyfill
